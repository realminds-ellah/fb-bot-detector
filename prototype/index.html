🔍 Bot / Troll Detector for Facebook

Interactive prototype — paste a comment section (or load the demo) and see the detector flag coordinated, bot-like, and AI-generated comments. Runs entirely in your browser.

Format per line: Author | comment | time — the author and time parts are optional.

How to copy comments from a real Facebook post
  1. Open the post on Facebook and scroll so the comments you want are visible (click “View more comments” to load more).
  2. Click and drag to select across the comments — from the first commenter’s name down through the last comment.
  3. Copy (⌘C / Ctrl+C).
  4. Tick the box above, paste here, and hit Analyze. The cleaner strips out “Like / Reply / 2h” noise automatically.

Best-effort: Facebook’s copied text is messy, so author/comment splitting won’t be perfect.

Prototype scope: this demonstrates the same on-device detection engine the browser extension uses (analyze.js), running on pasted/demo data. A web page can’t read live Facebook comments (browser security + FB login) — that’s what the extension is for. Findings are signals, not proof: the tool flags coordination patterns, not individual “this is a bot” verdicts.
